# TeoriaDosGrafos
Trabalho da faculdade de ciências da computação. 

Trabalho com duas funções, onde uma é a criação de centrais de rodoviaria de diferentes cidades e o cadastro de onibus ligando cidades com diferentes preços.
E também a criação de um mapeamento de um jogo de fps onde, são criados diferentes lugares e caminhos com diferentes tempos de locomoção.
